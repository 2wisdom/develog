## DNS 설정

---

DNS 설정은 도메인을 구입한 페이지에서 할 수 있다.

가비아의경우 My가비아 -> DNS 관리툴 -> DNS 관리 -> 설정해줄 도메인 주소의 DNS 설정에 들어가면 된다.

![dns-record](/posts/develop/2/dns-record.png)

DNS 레코드 추가를 하게되면 타입 / 호스트 / 값/위치 / TTL 을 적을 수 있는데,

아까 vercel 에 적혀있던 Type / Name / Value 를 순서대로 넣어주고

TTL은...

레코드 변경사항이 적용될 때까지 걸리는 시간(초)을 결정하는 DNS 레코드 값인데 보통 300이라고들 하는데 내 설정에는 600부터였다...

일단 가장 적은 시간인 600으로 채워넣었다. (이런 cs적인 부분들은 차차 알아가는걸로!)

![vercel-add-domain2](/posts/develop/2/vercel-add-domain2.png)

설정을 마치게되면 vercel 에서 등록이 완료되게 된다! (시간이 조금 걸린다)

그럼 이제 내가 구입한 도메인 [www.lizzie-develog.com](https://www.lizzie-develog.com/) 에 들어가면 내 블로그가 뜬다 히히

## 검색 등록

---

구글이나 네이버 검색에 웹페이지가 뜨게 하려면,

구글은 [Google Search Console](https://search.google.com/search-console/about), 네이버는 [Naver Search Advisor](https://searchadvisor.naver.com/)에 등록을 해야한다.

각각 등록하는 방법을 알아보자!

<br />

### [Google Search Console](https://search.google.com/search-console/about)

위 링크에 들어가서 시작하기를 누르면 속성 검색/추가가 있다.

![google-searchconsole1](/posts/develop/2/google-searchconsole1.png)

첫번째로 속성유형을 선택해주는데, 여기에 내 도메인주소를 넣어준다.

앞에 `https//` 랑 뒤에 `/` 는 모두 뺀다.

![google-searchconsole2](/posts/develop/2/google-searchconsole2.png)

그리고나면 DNS 레코드를 주는데, 이것도 아까 vercel 에서 등록해준것과 동일하게 넣어주면 된다.

type은 TXT / 호스트는 @ / 값에 TXT 레코드를 넣어주면 된다!

![google-searchconsole3](/posts/develop/2/google-searchconsole3.png)

설정을 하고나면 바로 등록되는것은 아닌데, 시간이 좀 걸린다.

들은바로는 산지 오래된 도메인은 며칠씩도 걸린다는데, 나는 방금 막 산 따끈따끈한 도메인이라 한 30분정도 걸린 것 같다.

![google-searchconsole4](/posts/develop/2/google-searchconsole4.png)

확인이 되면 짠!

이 창에서 내 웹페이지의 검색 노출 수나 클릭 수 등을 알아볼 수 있는 페이지가 생성이된다!
